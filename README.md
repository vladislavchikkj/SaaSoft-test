# Учетные записи (Account Manager)

Веб-приложение для управления учетными записями: добавление, редактирование и удаление записей с типами LDAP и Local.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Стек

Vue 3, TypeScript, Vite, Pinia, Tailwind CSS, reka-ui.
